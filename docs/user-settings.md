# 👥 User Settings

Per-user settings, managed individually — no admin access required.

## Account

Personal and security settings.

| Setting | Description |
|---|---|
| Account Name | Your display name |
| Two-Factor Authentication | Adds a second verification step at login. Recommended to protect your account from unauthorized access |
| Change Password | Update your password (up to 150 characters) |
| Language | English, Czech, German, Spanish, French, Italian, Russian, Portuguese (Brazil), Chinese (Simplified), Chinese (Traditional) — more may be added |
| Session Synchronization | How sessions sync: **across devices**, **same browser**, or **same tab** only |
| Passkeys | Create and manage passkeys |

## Appearance

Theme and layout preferences.

| Setting | Options |
|---|---|
| Theme | Dark, Light, Auto |
| Accent Color | Blue, Purple, Pink, Red, Orange, Green, Turquoise, Light Blue |
| Text Size | XS, S, M, L, XL |
| Custom Themes | Switch between or create custom CSS themes. Three built-in themes are managed via [Custom Sources](/customsource) |

## Terminal

Terminal look and feel.

| Setting | Options |
|---|---|
| Font | Font family and size |
| Cursor Style | Bar, Block, Underline |
| Cursor Blinking | Enable/disable |
| Theme | 12 built-in terminal color themes |

## File Manager

File and folder display behavior.

| Setting | Options |
|---|---|
| Default View Mode | Grid or List view on open |
| Image Thumbnails | Preview thumbnails in Grid view. Costs extra bandwidth to generate |
| Hidden Files | Show or hide dotfiles |
| Delete Confirmation | Require a confirmation prompt before deleting |
| Drag & Drop Action | Default behavior when dragging files: Move, Copy, or Ask each time |

## Keymaps

Shortcut bindings, each resettable to default.

| Action | Shortcut | Description |
|---|---|---|
| AI Menu | `CTRL + K` | Open the AI command menu |
| Broadcast Mode | `CTRL + B` | Toggle broadcast in split view |
| Copy | `CTRL + SHIFT + C` | Copy selection to clipboard |
| Full-Screen | `F11` | Toggle full-screen |
| Keyboard Shortcuts | `CTRL + SHIFT + K` | Open shortcut overview |
| Paste Identity Password | `CTRL + SHIFT + P` | Paste password of the active SSH identity |
| Quick Action | `CTRL + P` | Open quick action menu |
| Search | `CTRL + S` | Open search |
| Snippets | `CTRL + SHIFT + S` | Open snippets dialog |
